# healthTrack
This is a personal project inspired by my dad’s experience. He is diabetic and needs to record his glucose levels, but he often loses the paper where he writes them down. The idea is to create charts and reports that can be shared with his doctor to help monitor his glucose and blood pressure levels.

The web application is currently an MVP. In the future, it will also be developed as a mobile application.

The project was built using .NET for the API, SQLite as the database, and Angular for the frontend.
